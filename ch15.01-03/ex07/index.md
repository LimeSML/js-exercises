# クリックジャンキングとは
`iframe` を使ったクロスオリジンのページの埋め込みとユーザーによるクリックによって成立させる攻撃。
例えば、罠サイトの上に透明な `iframe` を配置して、意図せずに重要な操作をさせる。

# エラーとなる理由
同一オリジンポリシーにより、埋め込まれたページのオリジンと埋め込んでいる親ページのオリジンが異なれば、埋め込まれたページの DOM や Cookie、ストレージへの直接アクセはできない。

# 同一オリジンポリシーがない場合にどのような問題が発生しうるか
* セッションIDを取得し、不正ログインが可能になってしまう
* そもそもログインIDとパスワードが取得できる
* DOM 操作を行って重要な操作が可能となってしまう
